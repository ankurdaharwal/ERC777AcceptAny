# ERC777AcceptAny

This smart contract can be deployed in any Ethereum Chain.

To do so, we use the procedure described in [erc820](https://github.com/ethereum/EIPs/blob/master/EIPS/eip-820.md)

```
npm install
npm run build
node js/info.js
```

To Deploy it just send 0.01ETH to the `Sender` adderess and Broadcast the `RawTx`Transaction.

The contract will be deployed in the `Contract`address.
